#### Angular2 maps 

Yandex Maps API as an Angular2 direcitive.

### API

####Тег ya-map & ya-marker:
```html
 <ya-map [latitude]="55.76" [longitude]="37.64">
          <ya-marker [latitude]="55.847" [longitude]="38.6">
          </ya-marker>
           <ya-marker [latitude]="55.847" [longitude]="35.6">
          </ya-marker>
 </ya-map>
```
